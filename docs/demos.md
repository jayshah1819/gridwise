Interactive visualizations powered by Gridwise WebGPU primitives. demos run live in the browser and
require a WebGPU-capable browser (Chrome or Edge 113+, or Safari 18+).

## Interactive demos

### Galaxy Stars

A real-time particle simulation where GPU sort and scan primitives drive the position updates of thousands of stars. Click anywhere on the canvas to interact with the simulation.
